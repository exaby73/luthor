# Single Value Validators

Single Value Validators are used to validate any value that is not a map. See [Schema Validators][schema-validators] for validating maps.

Here is a simple example of how to use a single value validator to validate an email address:

```dart
import 'package:luthor/luthor.dart';

void main() {
  final emailValidator = l.string().email();
  print(emailValidator.validateValue("sample@example.com"));
  print(emailValidator.validateValue("not an email address"));
}
```

This will print:

```console
SingleValidationSuccess(data: sample@example.com)
SingleValidationError(data: not an email address, errors: [value must be a valid email address])
```
