# Luthor

Luthor is a pure [Dart][dart] validation library inspired by [Zod][zod]. With Luthor, you can create validation schemas and use them to validate data.

## Features

- [Single Value Validators][single-value-validators]: Luthor allows you to validate single (primitive) values like `String`, `int`, `double`, `bool`, `num`, `DateTime`, etc.
- [Schema Validators](/usage/schema-validators): Luthor allows you to validate complex data structures like `Map` and `List`.

[dart]: https://dart.dev
[zod]: https://zod.dev
