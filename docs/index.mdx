# Luthor

Luthor is a pure [Dart][dart] validation library inspired by [Zod][zod]. With Luthor, you can create validation schemas and use them to validate data.

## Features

- [Single Value Validators][single-value-validators]: Luthor allows you to validate single (primitive) values like `String`, `int`, `double`, `bool`, `num`, `DateTime`, and data structures like `List`.
- [Schema Validators](schema-validators): Luthor allows you to validate a schema. which is a `Map`.
