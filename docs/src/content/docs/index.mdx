## Start here

Luthor validates runtime input with small, composable validators. Use it for API payloads, forms, files, and nested data where you want typed results instead of thrown exceptions.

- [Install Luthor](/installation) and run your first validator.
- [Define schemas](/schemas/defining-schemas) for structured objects and nested data.
- [Generate schemas](/schemas/code-generation) from annotated Dart models.

## Common validators

Start with the scalar validators when you only need one value: [`string`](/single-values/string), [`int`](/single-values/int), [`double`](/single-values/double), [`bool`](/single-values/bool), [`list`](/single-values/list), and [`map`](/single-values/map).
